# Settlement Gate

Settlement Gate is an early-stage sentence-audit project.

Every completed sentence settles something. Meaning arrives quickly. The work here is to hold that moment open long enough to inspect the structure beneath the meaning.

The presentation page has two routes:

- Anthony: browser deck.
- Liam: speaker notes.
