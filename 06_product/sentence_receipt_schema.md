# Sentence Receipt Schema

This is an exploratory schema for a possible sentence receipt. It is not a final product specification.

```yaml
sentence:
settled_saying:
governing_object:
settlement_work:
range_scope:
governing_shape:
excluded_pull:
remainder:
orientation:
operator:
chosen_cell:
rival_pressure_check:
evidence_required:
action_permission:
review_status:
```

## Notes

The schema is intended to record what a sentence settles, what evidence it may require, and whether any action is permitted after review. Field names should be treated as early draft labels.
