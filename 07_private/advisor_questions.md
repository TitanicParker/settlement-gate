# Advisor Questions

## IP advisor

- What should be protected before wider disclosure?
- Should the method initially be treated as a trade secret?
- Is any part of the process patentable?
- What can safely be shown in a public demo?
- What records should be kept to document origin and development?

## AI / NLP reviewer

- Can the A-packet process be made reproducible?
- Can a resolver recover addresses without seeing the original sentence?
- What tests would distinguish structure from paraphrase?
- What model-to-model checks should be run first?

## Translation expert

- Do faithful translations preserve the proposed address?
- Which language pairs should be tested first?
- How should native-speaker review be structured?
- What counts as a translation shift rather than resolver error?

## Product builder

- What is the smallest useful prototype?
- Which workflow has the clearest user pain?
- What should a sentence receipt look like in practice?
- Where would this fit into existing AI or translation tools?

## Commercialisation mentor

- Which market should be tested first?
- Who is the first realistic buyer?
- What evidence would make the idea credible?
- What should not be disclosed before protection?

## Funder / investor

- What proof points are needed before funding?
- What is the defensible value?
- What risks need to be reduced first?
- What milestone would justify a larger build?

## University partner

- Which research group could pressure-test the claim?
- What validation study would be credible?
- Can this be tested without public overexposure?
- What academic or applied pathway is safest?
