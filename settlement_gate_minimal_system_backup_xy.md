# Settlement Gate: Minimal System Backup

A sentence is a settlement event.

It takes up something open, unstable, unresolved, unplaced, ungoverned, conditional, or merely available to thought. It performs structural work upon it. Attention resolves. A residue remains. That residue can then be inherited by another sentence, a reader, a proof, an institution, an AI system, or an action.

Settlement Gate is the system that makes this completion visible.

It begins with the sentence as given. It builds an A1-A6 address. It compresses the movement as X->Y. It recovers the three axes: Operator, Orientation, Remainder. It derives one of fifty-four completion cells. It issues a receipt. It permits stacking, where many sentences in the same cell saturate the reader with the structure beneath the content.

The sentence is the train.
The receipt discloses the tracks.
The stack makes the tracks visible.

The governing question is:

## 2. The A1-A6 Address

The A-packet is the sentence's address. It makes the sentence inspectable before it is placed in the completion space.

```text
A1 = Settled Saying
A2 = Governing Object
A3 = Settlement Work
A4 = Range / Scope
A5 = Governing Shape / Residue
A6 = Excluded Pull
```

### A1 — Settled Saying

A1 states the sentence's loyal ordinary survivor.

It answers:

```text
What is now said?
```

A1 keeps the source sentence present without allowing the source surface to dominate the structural resolution.

### A2 — Governing Object

A2 identifies what the sentence governs.

It answers:

```text
What object, condition, role, relation, field, demand, status, permission, action, or structure is governed?
```

A2 prevents the most visible noun from being mistaken for the governed object.

### A3 — Settlement Work

A3 names the work performed by the sentence.

It answers:

```text
What does the sentence do to what it governs?
```

A3 is the primary evidence for Operator.

### A4 — Range / Scope

A4 states where the settlement holds.

It answers:

```text
Where does this work hold, and what bounds its reach?
```

A4 is the primary evidence for Orientation.

### A5 — Governing Shape / Residue

A5 states what remains after the work has been performed.

It answers:

```text
What structural shape remains standing?
```

A5 is the primary evidence for Remainder.

### A6 — Excluded Pull

A6 names the rival pressure that must not govern.

It answers:

```text
What tempting rival reading, false continuation, local surface cue, or over-strong interpretation must remain defeated?
```

A6 is not a second candidate. It is the named pressure that the receipt keeps out of authority.

## 3. X->Y Traversal

The X->Y traversal is the compressed movement of the sentence.

```text
X = the uptake condition
Y = the landing configuration
```

X names what the sentence takes up.
Y names what the sentence leaves standing.
The arrow names the structural work by which X becomes Y.

Examples:

```text
unsettled account status -> active account status
candidate station field -> selected center station
open door-transition -> completed closing event
possible immediate send-action -> send-action blocked until approval exists
hidden sentence-work -> named and positioned movement-record
```

The traversal shows the shortcut that ordinary fluency hides.

## 5. Operator

Operator is the structural work by which the sentence becomes intelligible.

It answers:

```text
What kind of work stabilizes the survivor?
```

There are six Operators:

```text
Distinction
Expansion
Relation
Correction
Integration
Closure
```

### Distinction

Distinction makes something determinate.

It places, fixes, selects, assigns, limits, excludes, bounds, identifies, marks, or separates something as this rather than that.

Hidden form:

```text
This, not that.
This here.
This as this.
This bounded from that.
```

### Expansion

Expansion opens reach.

It opens a field, extends a direction, increases scope, continues beyond a boundary, multiplies availability, widens range, or makes outward continuation possible.

Hidden form:

```text
This may go further.
This opens beyond its prior limit.
This field becomes available.
This reach continues.
```

### Relation

Relation maintains bearing.

It stabilizes dependency, comparison, correspondence, support, recurrence, transfer, linkage, shared measure, passage, or structural answerability among terms.

Hidden form:

```text
This bears with that.
This depends on that.
This corresponds to that.
This carries through that.
This supports that.
```

### Correction

Correction alters a live prior reading.

It reverses, repairs, redirects, reweights, cancels, replaces, restores, corrects, or changes how something was being read.

Hidden form:

```text
Not as first read.
Not as formerly held.
The prior reading is repaired.
The former assignment is changed.
```

### Integration

Integration forms a working whole.

It organizes multiple parts, terms, stations, roles, cells, relations, or movements into a functional unit, body, system, arrangement, or field.

Hidden form:

```text
These parts form one working structure.
These terms are gathered into a unit.
This plurality becomes one body.
```

### Closure

Closure satisfies a live local demand.

It completes, preserves, licenses, validates, returns, fits, satisfies a condition, exhausts a requirement, or brings a process to rest.

Hidden form:

```text
The required thing is now satisfied.
The process has come to rest.
The condition is fulfilled.
The survivor remains despite disruption.
```

## 6. Orientation

Orientation is the direction attention takes after the Operator has been recognized.

It answers:

```text
Where does intelligibility resolve?
```

There are three Orientations:

```text
Toward
Across
Away
```

### Toward

Toward is focal convergence.

Attention resolves into a target, selected item, role, endpoint, formed unit, generated product, determinate placement, or focal state.

Hidden form:

```text
Attention lands here.
The sentence comes to this focal point.
The work resolves into this target.
```

### Across

Across is retained span.

Attention holds over a span, relation, comparison, route, matrix, field, condition-result structure, pair, dependency, or repeated case.

Hidden form:

```text
Attention holds between terms.
Attention traverses a field.
The sentence remains stretched across a structure.
```

### Away

Away is release from a prior frame.

Attention resolves by refusal, departure, release, preservation against, freedom from, movement beyond, or escape from an expected frame, false substitute, threat, confinement, dependency, or prior condition.

Hidden form:

```text
Not bound there.
Not replaced by that.
Not altered by that.
Beyond that limit.
Free from that frame.
```

---

## 7. Remainder

Remainder is what is left standing after the Operator has done its work and Orientation has resolved.

It answers:

```text
What mode of completion remains?
```

There are three Remainders:

```text
Is
With
Does
```

### Is

Is is standing status.

What remains is a state, identity, condition, availability, fixedness, placement, role, status, or fact.

Hidden form:

```text
Something now stands as the case.
```

### With

With is held bearing.

What remains is a relation, dependency, support, comparison, correspondence, pairing, balance, commonality, shared pattern, preservation-relation, or structurally remaining plurality.

Hidden form:

```text
Something remains with something else.
```

### Does

Does is enacted outcome.

What remains is an operation performed, result produced, structure formed, process completed, transformation made, field generated, passage enacted, or event brought about.

Hidden form:

```text
Something has been done.
```

## 8. Deriving the 54 Cells

The space is derived by combining:

```text
6 Operators x 3 Orientations x 3 Remainders = 54 cells
```

Each Operator has nine cells.

Each Operator cycles through:

```text
Toward / Is
Toward / With
Toward / Does
Across / Is
Across / With
Across / Does
Away / Is
Away / With
Away / Does
```

The cell output is:

```text
Operator / Orientation / Remainder
```

## 13. Stacking

Stacking places many receipts under the same cell.

A single sentence shows an instance.
A stack shows recurrence.
A saturated stack teaches the cell.

The stacking movement is:

```text
sentence 1 -> same cell
sentence 2 -> same cell
sentence 3 -> same cell
...
sentence 100 -> same cell
```

At first the reader sees content.
Then the reader sees repetition.
Then the reader sees the completion-form.

The purpose of stacking is to saturate attention with structure over content.
